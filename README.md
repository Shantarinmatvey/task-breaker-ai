# AI Task Breaker v2

🌍 **Онлайн-версия (Live Demo):** [https://Shantarinmatvey.github.io/task-breaker-ai/](https://Shantarinmatvey.github.io/task-breaker-ai/)

Умное веб-приложение для автоматической декомпозиции задач с использованием искусственного интеллекта (Google Gemini API). Приложение помогает планировать крупные проекты, разбивая сложные задачи на четкие, понятные и выполнимые шаги, организованные на удобной Kanban-доске.

## 🚀 Возможности

- **AI Декомпозиция**: Введите описание задачи (и опционально контекстные материалы/файлы), и встроенный ИИ автоматически разобьет ее на логические этапы.
- **Интерактивная Kanban-доска**: Удобное перемещение задач между колонками ("Сделать", "В процессе", "Готово") с помощью drag-and-drop.
- **Поддержка файлов и контекста**: Возможность прикреплять текстовые файлы, документы (.docx, .pdf) и изображения, чтобы ИИ учитывал все нюансы вашей задачи.
- **Синхронизация через Firebase**: Надежное облачное хранение всех проектов и задач, привязанных к вашему аккаунту. 
- **Безопасное хранение API ключа**: API-ключ Gemini сохраняется исключительно локально в браузере (LocalStorage) и никуда не передается, кроме серверов Google при выполнении запроса.
- **Современный UI**: Отзывчивый, красивый интерфейс с плавной анимацией и поддержкой современных веб-стандартов.

## 🛠 Технологии

Приложение построено с использованием нативного веб-стека (Vanilla JS, HTML5, CSS3) и не требует сборки.

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Стилизация**: Custom CSS (переменные, flexbox, grid, glassmorphism)
- **Backend & Auth**: Google Firebase (Firestore для БД, Firebase Auth для аутентификации)
- **ИИ Интеграция**: Google Gemini 1.5 Pro/Flash
- **Библиотеки**: 
  - `Sortable.js` (для Kanban drag-and-drop)
  - `Mammoth.js` (для парсинга DOCX файлов)
  - `FontAwesome` (для иконок)

## ⚙️ Установка и запуск

Поскольку это клиентское веб-приложение без этапа сборки, его можно запустить очень просто:

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/ВАШ_НИКНЕЙМ/task-breaker-ai.git
   cd task-breaker-ai
   ```

2. Запустите локальный веб-сервер. Вы можете использовать любой HTTP-сервер, например:
   - Если установлен Python: `python -m http.server 3000`
   - Если установлен Node.js: `npx serve .` или `npx http-server`
   - Или использовать расширение "Live Server" в VS Code.

3. Откройте приложение в браузере (например, `http://localhost:3000`).

## 🔑 Настройка API-ключа (Для работы ИИ)

Чтобы функция автоматического разбиения задач работала, каждому пользователю необходимо указать свой API-ключ Gemini:

1. Зарегистрируйтесь / войдите в аккаунт в приложении.
2. В левом нижнем углу нажмите **"Настройки API"** (иконка ключа).
3. Получите бесплатный ключ в [Google AI Studio](https://aistudio.google.com/app/apikey).
4. Вставьте ключ в поле и нажмите "Сохранить".

*(Ключ сохраняется в памяти вашего браузера).*

## 🔒 Безопасность и Firebase
Для полноценной работы проекта в вашем форке потребуется настройка собственного Firebase:
1. Создайте проект в Firebase Console.
2. Включите **Authentication** (Email/Password) и **Firestore Database**.
3. Замените `firebaseConfig` в файле `auth.js` на данные вашего проекта.
